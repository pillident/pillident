
# Project Blueprint - Pillident (알약 식별 서비스)

## Overview

Pillident은 모던 웹 기술(Web Components, CSS Layer/Variables, ES Modules)을 활용하여 사용자가 알약 사진을 촬영하거나 업로드하여 쉽고 신속하게 알약을 식별할 수 있도록 돕는 웹 애플리케이션입니다. 

## Design and Features

### Aesthetics & Style
*   **Color Palette:** 깨끗하고 현대적인 느낌을 주는 밝은 회색, 산뜻한 파란색, 그리고 흰색을 중심으로 한 라이트 테마.
*   **Background:** 부드러운 화이트/라이트 블루 그라데이션과 밝은 회색 배경을 사용하여 화사하고 편안한 시각적 경험 제공.
*   **Visual Effects:** 요소들의 입체감을 살리는 부드러운 그림자(Soft shadow)와 파란색 글로우(Glow) 효과로 상호작용 포인트를 강조.
*   **Iconography:** 사용자 직관성을 돕는 정교하고 아름다운 SVG 아이콘 활용.

### Features
1.  **메인 히어로 세션:** 알약 식별의 목적을 나타내는 멋진 타이포그래피 타이틀과 설명.
2.  **알약 식별 컴포넌트 (`<pill-identifier>`):**
    *   **사진 업로드/촬영 활성화 버튼:** 모바일 카메라 연동 및 이미지 선택 창을 열어주는 감각적인 글로우 버튼.
    *   **이미지 프리뷰 영역:** 선택된 이미지를 깔끔한 둥근 카드 형태로 배치.
    *   **스캔 레이저 애니메이션:** 이미지가 로드되면 자동으로 세련된 일렉트릭 블루 색상의 스캐닝 레이저 라인이 위아래로 움직이며 분석 중인 느낌의 고품질 피드백 선사.
    *   **AI 분석 시뮬레이션:** '분석 시작' 버튼 클릭 시, Gemini API 연동을 시뮬레이션하여 가상 데이터베이스 기반으로 결과를 동적으로 표시.
3.  **분석 결과 컴포넌트 (`<pill-result>`, `<pill-references>`):**
    *   **결과 카드:** 식별된 알약의 이미지, 이름, 성분, 제조사를 명확하게 표시.
    *   **참고 자료:** 식별된 알약에 대한 설명, 참고 이미지, Google 이미지 검색으로 연결되는 외부 링크를 제공하여 신뢰도 향상.

## Current Plan: V4 - Real AI Integration (Gemini Simulation)

**Objective:** 실제 Gemini와 같은 멀티모달 AI API를 연동하기 위한 기반을 다집니다. AI 분석 로직을 별도의 모듈(`ai.js`)로 분리하고, 실제 API 호출과 유사한 비동기(async/await) 방식으로 코드를 리팩토링하여, 향후 실제 API 키만 추가하면 바로 작동할 수 있는 구조를 만듭니다.

**Steps:**
1.  **[Done]** AI 분석 로직을 담당할 `ai.js` 파일을 새로 생성했습니다.
2.  **[Done]** `main.js` 파일에서 `setTimeout`으로 구현된 분석 시뮬레이션 부분을 제거했습니다.
3.  **[Done]** `ai.js`의 `analyzePillImage` 함수를 `import`하여 호출하는 방식으로 변경했습니다. 
4.  **[Done]** 브라우저 프리뷰에서 분석 과정이 이전과 동일하게 잘 작동하는지 확인했습니다.
